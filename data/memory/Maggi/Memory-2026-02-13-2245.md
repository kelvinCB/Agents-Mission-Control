# Feature: Implementar Login Frontend - VPS Visual Dashboard

## 🎯 Objetivo
Implementar un flujo de **login en el frontend** para el **VPS Visual Dashboard**, de modo que solo usuarios autenticados puedan acceder a las vistas protegidas del dashboard.

## 📋 Problemática
Actualmente el VPS Visual Dashboard permite acceso sin autenticación (o el flujo es incompleto), lo que:
- expone funcionalidades/datos a usuarios no autorizados,
- rompe el flujo esperado de seguridad del producto,
- dificulta el control de sesión/usuario y la personalización.

- **Unit tests (Frontend)**:
  - Render del formulario de login
  - Validaciones de campos (email/username/password) y mensajes de error

- **Integration tests**:
