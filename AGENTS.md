# AGENTS.md — Agents Mission Control

## Mandatory Frontend UI/UX Standard

For any frontend/UI/UX work in this repository, **Uncodixfy is mandatory**.

- Skill path: `.agents/skills/uncodixfy/SKILL.md`
- Rules file: `.agents/skills/uncodixfy/Uncodixfy.md`
- Command trigger (supported agents): `/uncodixfy`

## Enforcement Rule

Before generating or editing frontend files (`frontend/src/**`, CSS, components, pages):
1. Load/apply Uncodixfy guidance.
2. Avoid default AI UI patterns (glassmorphism, decorative gradients, oversized radius, hero-like dashboard blocks, ornamental labels).
3. Keep UI strict, functional, and predictable (clean hierarchy, subtle borders/shadows, consistent spacing).

If an output conflicts with Uncodixfy, refactor it before finalizing.
